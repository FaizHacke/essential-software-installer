# 🚀 Essential Software Installer

**One command to install 25 essential apps on Windows & Linux**

## 📦 What Gets Installed

### Windows (16 apps)
- Microsoft PowerToys, Everything, 7-Zip
- Firefox, Discord, Telegram
- Git, VS Code, Docker Desktop
- VLC, GIMP, OBS Studio
- LibreOffice, Bitwarden
- FileZilla, Microsoft PC Manager

### Linux (18 apps)
- Firefox, LibreOffice, VLC, GIMP
- Git, Docker, VS Code
- Timeshift, Stacer, BleachBit
- tmux, htop, Flameshot
- KeePassXC, Discord, Telegram
- OBS Studio, FileZilla

## 🎯 Quick Start

### Windows (PowerShell as Administrator)
```powershell
irm https://raw.githubusercontent.com/YOUR_USERNAME/essential-software-installer/main/install_windows.ps1 | iex

Linux (Ubuntu/Debian)
curl -fsSL https://raw.githubusercontent.com/YOUR_USERNAME/essential-software-installer/main/install_linux.sh | bash

Manual Installation
Clone and run locally:
git clone https://github.com/YOUR_USERNAME/essential-software-installer.git
cd essential-software-installer
