# Contributing to Essential Software Installer

Thank you for your interest! Here's how you can help:

## 🐛 Reporting Issues

- Check if the issue already exists
- Include your OS version and error messages
- Be specific about which software failed to install

## 💡 Suggesting New Software

To suggest adding an app, please include:
- App name and why it's "essential"
- Windows: `winget` ID
- Linux: package names for apt/dnf/pacman

## 🔧 Pull Requests

1. Fork the repository
2. Create a feature branch
3. Test your changes on both Windows and Linux
4. Submit a pull request with clear description

## 📝 Guidelines

- Keep the focus on **essential** software (tools most people need)
- Prioritize open-source and free software
- Test your changes before submitting
